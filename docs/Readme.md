## How to use the API docs...

Please Import the `API Docs [Insomnia v5].yaml` file into your local installation of [Insomnia API Client](https://insomnia.rest/download).

Compatible with Insomnia v5+.

### Notes for local use.

- After importing the collection, execute the `Login` endpoint first with a valid credentials to populate the coookie-jar. This will allow for further requests to be authenticated automatically.
